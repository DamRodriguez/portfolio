# Project Overview

This is my personal portfolio.

# Project Guidelines

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion and GSAP for animations
- next-intl for internationalization
- React Hook Form and Zod for form validations
- React Toastify for notifications

## Code Style

- Use TypeScript strictly.
- Avoid using `any`.
- Prefer functional components.
- Keep components small and reusable.
- Use descriptive variable names.
- Always modularized components.

## Styling

- Use Tailwind CSS only.
- Do not create CSS files unless explicitly requested.
- Reuse existing utility classes whenever possible.

## Project Structure

- Components go in `/components`.
- Hooks go in `/hooks`.
- Constants go in `/constants`.

## Animations

- Use Framer Motion for entry and others animations.
- Use GSAP for scroll animations.
- Prefer smooth, subtle animations.
- Reuse existing motion components when possible.

## Before Writing Code

- Reuse existing components whenever possible.
- Follow the current project architecture.
- Do not introduce new libraries unless requested.
